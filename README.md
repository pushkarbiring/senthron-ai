# 🚀 Senthron – Emotion-Aware AI System

Senthron is an AI-based system designed to analyze human emotions from input data (text/voice) and generate emotionally adaptive responses.

This project works in 3 main stages:
1. Data Collection  
2. Model Training  
3. Live Execution  

---

## 📂 Project Structure

```
.
├── 01_data_harvester.py     # Collects and prepares dataset
├── 02_train_senthron.py     # Trains the AI model
├── 03_senthron_live.py     # Runs the system in real-time
```

---

## ⚙️ Requirements

- Python 3.10+
- pip

### Install dependencies:
```bash
pip install -r requirements.txt
```

*(If requirements.txt is not available, install libraries manually based on imports.)*

---

## ▶️ How to Run

Run the files **in order**:

### 1️⃣ Data Collection
```bash
python 01_data_harvester.py
```

### 2️⃣ Model Training
```bash
python 02_train_senthron.py
```

### 3️⃣ Live Execution
```bash
python 03_senthron_live.py
```

---

## 🧠 Features

- Emotion detection (text/voice)
- Real-time response system
- Modular pipeline (data → training → live)
- Easy to expand and improve

---

## 📌 Important Notes

- Always run scripts in sequence
- Run training only after data collection
- Run live system only after training is complete
